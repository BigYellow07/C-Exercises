# 公共练习区：第一周作业  

## 1【入门】 简单的 printf 和 scanf编程练习  
  
printf函数和scanf函数是C语言中最常用的函数之一，下面让我们练习一下吧。  
  
输入：  
    整型变量x的值。  
  
输出：  
    输出整型变量x对应的十进制、八进制和十六进制形式。  
  
认真看看 -1 的八进制和十六进制输出，请想想是为什么。  

## 输入输出样例 #1

### 输入 #1

```
1
```

### 输出 #1

```
x=1,x=1,x=1
```

## 输入输出样例 #2

### 输入 #2

```
10
```

### 输出 #2

```
x=10,x=12,x=a
```

## 输入输出样例 #3

### 输入 #3

```
-1
```

### 输出 #3

```
x=-1,x=37777777777,x=ffffffff
```
